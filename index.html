Introduction

GitHub

Welcome to Luxury UI, the most sophisticated, high-performance, and visually stunning user interface framework for Roblox.

Luxury UI is engineered from the ground up using custom Luau objects to deliver absolute smoothness, flawless animations, and high-fidelity responsiveness. Designed around a sleek obsidian black and brushed metal theme, it provides developers and script hubs with an interface that looks premium and loads instantaneously.

Core Features

  • Absolute Luxury Theme: Deep charcoal and obsidian gradients, detailed thin strokes, glowing state-indicators, and elegant gold or indigo accenting.
  • Sleek Draggable Windows: Draggable title bars, custom minimizations, and close functions.
  • Rich Element Catalog: Over 10 fully interactive components including collapsible dropdowns, smooth slider tracks, textboxes, keybind listeners, and presets-based colorpickers.
  • Dynamic Toasts: Queueable, rich notification toasts with duration-based progress bars.
  • Dual-Mode Bootstrapper: Runs perfectly in Roblox Studio via standard PlayerGui, or bypasses detection inside CoreGui within injector environments.

Interactive Demo

You can interact with the **Live Simulator** on the right side of the screen! Click on elements in the simulated screen, or use the interactive guide buttons inside the API reference to trigger animations and states in real-time.

Quick Start & Installation

To run Luxury UI in your script, simply copy and paste the loader bootstrapper below. It dynamically fetches the library script, creates a new execution context, and loads the user interface framework.

Loadstring Bootstrapper

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/Varomine/Roblox-UI/refs/heads/main/luxury_library.lua"))()

local Window = Library:CreateWindow({
    Name = "Luxury Hub",
    Theme = "Gold" -- "Gold" or "Indigo"
})

Development Mode (Roblox Studio)

If you are writing scripts in Roblox Studio and want to load Luxury UI locally, place the `luxury_library.lua` code inside a `ModuleScript` named `LuxuryUI` under `ReplicatedStorage` and load it via `require`:

local Library = require(game:GetService("ReplicatedStorage"):WaitForChild("LuxuryUI"))

local Window = Library:CreateWindow({
    Name = "Luxury Hub",
    Theme = "Gold"
})

Themes & Styling

Luxury UI supports programmatically switching color systems. Out of the box, we support two luxury themes: **Gold** (Default Obsidian & Satin Gold) and **Indigo** (Deep Indigo Accent).

Gold Theme

The flagship luxury palette. Solid blacks, gold metallic accents, and high contrast typography. Best for VIP dashboards.

Indigo Theme

A modern cybernetic variant. Deep dark blues, neon-purple edges, and smooth violet hover glowing fields.

Theme Configuration

You can change the starting theme in your `CreateWindow` configuration object:

local Window = Library:CreateWindow({
    Name = "Luxury Hub",
    Theme = "Indigo" -- Changes primary colors and accents to Indigo
})

Library:CreateWindow

Initializes the ScreenGui and returns a new Window object. This is the main dashboard frame that will contain your tabs and options.

Code Sample

local Window = Library:CreateWindow({
    Name = "Luxury Hub",
    Theme = "Gold" -- "Gold" or "Indigo"
})

Parameters

Property Type Default Description
Name string "Luxury Hub" Title displayed in the top-left logo and title card.
Theme string "Gold" Color theme configuration. Either "Gold" or "Indigo".

Window:CreateTab

Creates a navigation page within the window. Tabs are represented in the left-hand sidebar with a custom icon and label.

The first tab created is automatically selected and shown by default when the user interface loads.

Code Sample

local Tab = Window:CreateTab({
    Name = "Combat",
    Icon = "rbxassetid://10888374828" -- Roblox Image Asset ID
})

Parameters

Property Type Default Description
Name string "Tab" Label displayed in the navigation menu and topbar.
Icon string "rbxassetid://10888374828" Roblox Image ID of the icon to display next to the tab label.

Tab:CreateSection

Groups elements within a tab. It adds a category header in all caps and a thin styling line next to it to partition elements logically.

Code Sample

local Section = Tab:CreateSection("Target Settings")

Parameters

Property Type Default Description
Name string "Section" Label content of the group heading.

Tab:CreateLabel

Creates a static text label component inside the active tab. Labels are highly dynamic; using the returned controller's :Set() method, they can be updated live at runtime to display stats, server time, player count, ping, or toggle states.

Code Sample

-- Create a text label
local StatusLabel = Tab:CreateLabel({
    Name = "Status: Idle"
})

-- Or pass a string directly for default config:
local FpsLabel = Tab:CreateLabel("FPS: 60")

-- Update the text label dynamically later (e.g. in a loop)
StatusLabel:Set("Status: Farming Bone...")

Parameters

Property Type Default Description
Name / Configuration string or table "Label" The text to display on the label, or a configuration table containing a Name field.

Returned Object Methods

Method Arguments Description
Set newVal: string Updates the label's text dynamically on screen.

Tab:CreateToggle

Creates a toggle switch widget that switches between ON (true) and OFF (false) states. Best used for enabling/disabling loops or persistent cheats (e.g. Aimbot, ESP).

Code Sample

local Toggle = Tab:CreateToggle({
    Name = "Aimbot Toggle",
    CurrentValue = false,
    Callback = function(Value)
        print("Aimbot state: ", Value)
    end
})

-- Change state programmatically
Toggle:Set(true)

Parameters

Property Type Default Description
Name string "Toggle" The text to display next to the switch.
CurrentValue boolean false Starting visual and logical state of the toggle.
Callback function nil Runs when the state changes. Returns the new boolean state.

Tab:CreateSlider

Creates a slider bar that allows the user to drag a handle to choose a numeric value within a specific range.

Code Sample

local Slider = Tab:CreateSlider({
    Name = "Aimbot FOV",
    Min = 0,
    Max = 180,
    CurrentValue = 90,
    Callback = function(Value)
        print("FOV Value: ", Value)
    end
})

-- Set value programmatically
Slider:Set(120)

Parameters

Property Type Default Description
Name string "Slider" The text to display.
Min number 0 Minimum drag value bounds.
Max number 100 Maximum drag value bounds.
CurrentValue number 50 Starting value.
Callback function nil Fires continuously while dragging or when set. Returns the current number.

Tab:CreateDropdown

Creates a select menu containing multiple options. Clicking the menu expands it with a clean sliding animation, letting the user select a single active option.

Code Sample

local Dropdown = Tab:CreateDropdown({
    Name = "Aim Bone Target",
    Options = {"Head", "Torso", "Left Arm", "Right Arm"},
    CurrentOption = "Head",
    Callback = function(Option)
        print("Target part: ", Option)
    end
})

-- Set value programmatically
Dropdown:Set("Torso")

Parameters

Property Type Default Description
Name string "Dropdown" Text label to show.
Options table {} Array list of string choices.
CurrentOption string "" Starting active choice.
Callback function nil Fires when the option is changed. Returns the selected string choice.

Tab:CreateTextBox

Creates a text box component that accepts typed keyboard inputs. Fires the callback once the user finishes typing and presses enter or clicks away.

Code Sample

local TextBox = Tab:CreateTextBox({
    Name = "WalkSpeed Override",
    Placeholder = "Enter value...",
    Callback = function(Text)
        local Num = tonumber(Text)
        if Num then
            print("Set speed value: ", Num)
        end
    end
})

-- Set text programmatically
TextBox:Set("100")

Parameters

Property Type Default Description
Name string "Text Input" Label of the input frame.
Placeholder string "Type here..." Hint text shown when the input box is empty.
Callback function nil Fires on enter key or focus lost. Returns the string content.

Tab:CreateKeybind

Binds a key on the keyboard to trigger a specific callback. Users can click the keybind box to re-bind it to any key dynamically.

Code Sample

local Keybind = Tab:CreateKeybind({
    Name = "Self Destruct Bind",
    CurrentKeybind = "Delete",
    Callback = function(Key)
        print("Self destruct activated!")
    end
})

Parameters

Property Type Default Description
Name string "Keybind" The text to show.
CurrentKeybind string "None" Starting bound key name (e.g. "RightControl", "F").
Callback function nil Fires when the key is pressed. Returns the KeyCode object.

Tab:CreateColorpicker

Adds a customizable color selector. Clicking it drops down a panel containing standard presets of colors that can be selected.

Code Sample

local Colorpicker = Tab:CreateColorpicker({
    Name = "Chamber Glow Color",
    Default = Color3.fromRGB(212, 175, 55),
    Callback = function(Color)
        print("Selected Color RGB: ", Color.R, Color.G, Color.B)
    end
})

-- Set color programmatically
Colorpicker:Set(Color3.fromRGB(255, 0, 0))

Parameters

Property Type Default Description
Name string "Colorpicker" Label of the picker.
Default Color3 Color3.fromRGB(255, 255, 255) Default starting color.
Callback function nil Fires when color is picked. Returns the selected Color3 object.

Library:Notify

Triggers a floating notification toast. These toasts appear in the bottom-right corner of the screen, slide in smoothly, and fade away automatically after their duration concludes.

Code Sample

Library:Notify({
    Title = "Luxury UI Engine",
    Content = "The script loaded successfully inside 0.14 seconds.",
    Duration = 5,
    Icon = "rbxassetid://10888373307" -- Bell Asset ID
})

Parameters

Property Type Default Description
Title string "Notification" Main bold title of the notification card.
Content string "" Details and descriptions of the event.
Duration number 4 Time in seconds before the notification closes.
Icon string "http://www.roblox.com/asset/?id=6023426923" Ignored. Always displays the custom military badge icon (user cannot change).
